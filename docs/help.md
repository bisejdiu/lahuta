# Learning & Getting help

If you need help getting started with Lahuta or with advanced usage, the following sources may be useful.

## Tutorials

The [tutorials](tutorials/getting_started.md) provide a step-by-step guide to using Lahuta. It is recommended for quickly learning the basics. 

## Usage Documentation

The best place to start learning how to use Lahuta is the [usage documentation](usage/intro.md), which includes a comprehensive overview of the library's functionality along with technical details and examples.

## API Documentation

Refer to the [API documentation](api/core/luni.md) for detailed references encompassing all publicly accessible Lahuta APIs.

## GitHub Discussions

You can also use [GitHub discussions](https://github.com/bisejdiu/lahuta/discussions) for asking questions, getting help, and sharing ideas.
