# lahuta

# Installation
Ideally, please use conda

```bash
conda install -c conda-forge openbabel mdanalysis
cd lahuta
pip install -e . 
```

When there are new changes in GitHub, you can update your version simply bu running: 
```bash
git pull origin main
```
